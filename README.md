# 🚀 AI & Data Engineering Full-Stack Course
> **데이터 엔지니어링부터 LLM 기반 AI 서비스 배포까지의 실무 파이프라인 구축**

이 저장소는 **SK Family AI Bootcamp**에서의 학습 기록과 프로젝트를 정리한 공간입니다. 단순한 이론 학습을 넘어 데이터 수집, DB 설계, 모델링, 서비스 배포까지의 Full-Stack 역량을 기르는 데 집중했습니다.

---

## 🛠 Tech Stacks
- **Languages:** Python (3.12+), SQL, HTML/JS
- **Data Eng:** MySQL, Pandas, BeautifulSoup, Selenium, Regex
- **AI/ML/DL:** Scikit-learn, PyTorch, OpenCV, SVD(Matrix Decomposition)
- **LLM:** LangChain, OpenAI API, Vector DB (Chroma, FAISS)
- **DevOps:** Docker, Streamlit, uv (Fast Package Manager)

## 📂 Curriculum & Key Learning Sections

### 1️⃣ 데이터 엔지니어링 및 관리
* **MySQL을 활용한 ERD 설계 및 파이썬 연동**을 통해 효율적인 데이터 수집·조회 시스템을 구축했습니다.
* 관계형 데이터베이스의 정규화와 인덱싱, 파이썬을 이용한 대량 데이터 적재 프로세스를 익혔습니다.


### 2️⃣ 머신러닝/딥러닝 모델링
* **탐색적 데이터 분석(EDA)** 기반 데이터 전처리 및 비즈니스 목표에 최적화된 최적 모델 선정·배포 프로세스를 수행했습니다.
* Scikit-learn을 이용한 머신러닝 알고리즘 적용과 PyTorch를 활용한 딥러닝 모델 설계를 실습했습니다.


### 3️⃣ LLM 및 RAG 시스템 구현
* **LangChain**을 활용하여 벡터 데이터베이스와 LLM을 연동한 질의응답(**RAG**) 서비스 개발 및 프롬프트 엔지니어링을 수행했습니다.
* 외부 지식 베이스를 활용하여 LLM의 신뢰성을 높이는 지능형 시스템 구축 역량을 확보했습니다.


### 4️⃣ Full-stack AI 앱 배포
* 클라우드 인프라를 기반으로 **프론트엔드(HTML/JS), 백엔드 API, DB 및 LLM을 통합**한 웹 서비스 구축 및 배포를 완료했습니다.
* Docker를 활용한 컨테이너화 전략으로 서비스의 이식성과 확장성을 확보했습니다.

### 5️⃣ 실무 멘토링 프로젝트
* 현업 전문가 피드백을 반영하여 **기업 특화 sLLM 챗봇** 등 실제 비즈니스 시나리오를 해결하는 프로젝트를 수행 중입니다.
* 실제 데이터와 도메인 지식을 결합하여 실질적인 비즈니스 가치를 창출하는 모델을 개발합니다.
