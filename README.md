## 🏆 Featured Projects

### ⚾ KBO Data Pipeline (Data Engineering)
**[🔗 GitHub Repository](https://github.com/bizseohyunkim/KBO-Data-Pipeline) | [📓 Project Dev Log](https://github.com/bizseohyunkim/KBO-Data-Pipeline/blob/main/KBO_%EB%8D%B0%EC%9D%B4%ED%84%B0_%ED%8C%8C%EC%9D%B4%ED%94%84%EB%9D%BC%EC%9D%B8_%ED%94%84%EB%A1%9C%EC%A0%9D%ED%8A%B8_%EA%B0%9C%EB%B0%9C_%EC%9D%BC%EC%A7%80.ipynb)**

> 데이터 수집부터 SQL 적재까지의 전 과정을 자동화한 **ETL 파이프라인** 구축 프로젝트입니다.

* **Data Collection**: Selenium/Requests 기반 동적 크롤링 자동화 (약 900여 명의 선수 데이터 수집)
* **Data Processing**: 정규표현식(Regex)을 활용한 비정형 데이터(연봉, 신체정보) 전처리 및 BMI 지수 산출
* **DB Management**: MySQL ERD 설계 및 Window Function(`RANK`)을 활용한 구단별 데이터 분석 수행

## 📂 Curriculum & Key Learning Sections

### 1️⃣ 데이터 엔지니어링 및 관리
* **MySQL을 활용한 ERD 설계 및 파이썬 연동**을 통해 효율적인 데이터 수집·조회 시스템을 구축했습니다.
* 관계형 데이터베이스 아키텍처 이해 및 파이썬 기반 데이터 적재(ETL) 자동화 프로세스를 익혔습니다.


[Image of an Entity-Relationship Diagram for a database]


### 2️⃣ 머신러닝/딥러닝 모델링
* **탐색적 데이터 분석(EDA)** 기반 데이터 전처리 및 비즈니스 목표에 최적화된 최적 모델 선정·배포를 수행했습니다.
* Scikit-learn 및 PyTorch를 활용하여 정형/비정형 데이터 모델링과 SVD(특이값 분해) 이미지 압축 알고리즘을 구현했습니다.


### 3️⃣ LLM 및 RAG 시스템 구현
* **LangChain**을 활용하여 벡터 데이터베이스와 LLM을 연동한 질의응답(**RAG**) 서비스 개발 및 프롬프트 엔지니어링을 수행했습니다.
* 외부 지식 베이스를 결합하여 할루시네이션을 방지하는 지능형 AI 에이전트 구축 역량을 확보했습니다.


### 4️⃣ Full-stack AI 앱 배포
* 클라우드 인프라를 기반으로 **프론트엔드(HTML/JS), 백엔드 API, DB 및 LLM을 통합**한 웹 서비스 구축 및 배포를 완료했습니다.
* Docker 컨테이너 기반의 배포 전략으로 개발 환경과 운영 환경의 일관성을 유지했습니다.

### 5️⃣ 실무 멘토링 프로젝트
* 현업 전문가 피드백을 반영하여 **기업 특화 sLLM 챗봇** 등 실제 비즈니스 시나리오를 해결하는 프로젝트를 수행 중입니다.
* 비즈니스 요구사항 분석부터 실제 배포까지 AI 서비스의 전체 생명주기를 경험했습니다.
