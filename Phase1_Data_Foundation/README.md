# 📊 Phase 1: Data Foundation 학습 기록

데이터 분석의 기초가 되는 Python 프로그래밍, 데이터 가공, 그리고 효율적인 데이터 관리 시스템(MySQL) 구축을 학습합니다.

### 🗓️ 1~2주차 학습 요약 (Foundation)
* **2026-01-02 (1주차)**: 캠프 오리엔테이션 및 개발 환경 세팅
* **2026-01-05 ~ 01-08 (2주차)**: 
    * **Pandas**: 데이터 로드, 결측치 처리 및 데이터 가공 실습
    * **정규표현식**: 비정형 텍스트 데이터 추출 및 클리닝 기법 마스터
    * **Streamlit**: 파이썬 기반 웹 대시보드 제작 및 보안 설정(Secrets) 실습
* **2026-01-09**: **MySQL** 서버 구축 및 로컬/원격 네트워크 설정

### 🛠️ 주요 기술 스택
- **Language**: Python
- **Library**: Pandas, NumPy, Streamlit
- **Database**: MySQL Workbench, SQL Server
